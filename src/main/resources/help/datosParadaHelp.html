Ayuda - Datos de Parada

En esta ventana podrás visualizar y gestionar las estancias de los peregrinos en la parada actual, dentro de un rango de fechas específico.

Campos de Filtro

  • Fecha Inicio: Permite seleccionar la fecha de inicio del rango para consultar las estancias.
  • Fecha Fin: Permite seleccionar la fecha de fin del rango para consultar las estancias.
  • ID de Parada: Muestra el identificador único de la parada. Este campo se llena automáticamente al buscar los datos.
  • Nombre de Parada: Muestra el nombre de la parada. Este campo se llena automáticamente al buscar los datos.
  • Región: Muestra la región de la parada. Este campo se llena automáticamente al buscar los datos.

Tabla de Estancias

La tabla muestra la información relacionada con las estancias de los peregrinos en el rango de fechas seleccionado.

  • ID: Identificador único de la estancia.
  • Peregrino: Nombre del peregrino asociado a la estancia.
  • Fecha: Fecha en que se registró la estancia.
  • VIP: Indica si la estancia es de tipo VIP (sí/no).

Acciones

  • Buscar: Al hacer clic en este botón, se realizaría la consulta según el rango de fechas introducido y se mostrarán los resultados en la tabla de estancias.

Validación de Fechas

Las fechas seleccionadas deben ser correctas. Si alguna de las siguientes condiciones no se cumple, se mostrará un mensaje de error:

  • La fecha de inicio no puede ser posterior a la fecha actual.
  • La fecha de fin no puede ser posterior a la fecha actual.
  • La fecha de inicio no puede ser posterior a la fecha de fin.

Errores Comunes

  • Fechas Incorrectas: Si las fechas seleccionadas no son válidas, aparecerá una alerta con el mensaje correspondiente.
  • Parada no encontrada: Si no se pueden obtener las estancias, verifica que la parada esté correctamente configurada y que se haya asignado un ID válido.

Consejos

  • Asegúrate de que el rango de fechas esté dentro de los límites permitidos y que las fechas sean coherentes.
  • Verifica que la parada a la que estás consultando las estancias es la correcta.
  • Si los datos no se cargan correctamente, intenta refrescar la página o comprobar la conectividad con la base de datos.
