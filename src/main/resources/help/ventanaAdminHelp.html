Ayuda - Ventana de Administrador

En esta ventana, el administrador puede gestionar diferentes opciones del sistema, como registrar paradas y consultar estadísticas de los peregrinos.

Campos de Información

  • Número de Peregrinos: Muestra el total de peregrinos registrados en el sistema.
  • Número de Paradas: Muestra el total de paradas disponibles en el sistema.
  • Peregrino Activo: Aquí se indicaría el peregrino más activo, basado en la información disponible del sistema (aún no implementado).
  • Parada Frecuente: Esta sección también es para mostrar la parada más frecuente, aunque aún no ha sido implementada.

Botones de Acción

  • Registrar Parada: Redirige al formulario para registrar una nueva parada en el sistema.
  • Cerrar Sesión: Permite cerrar la sesión del administrador y volver a la ventana de inicio de sesión.

Menú de Acciones Rápidas

  • Ctrl+R: Atajo para abrir la ventana de registro de parada.
  • Ctrl+A: Atajo para abrir esta ventana de ayuda.
  • Ctrl+S: Atajo para cerrar la sesión del administrador.

Errores comunes

  • Si alguna de las opciones no responde, asegúrate de que los servicios asociados estén correctamente configurados.
  • Si al intentar registrar una parada el sistema no responde, revisa la conexión con el servicio de paradas.
  • Si al intentar cerrar sesión no sucede nada, verifica si la sesión ya estaba cerrada o si hay problemas con el servicio de sesión.

Consejos

  • Asegúrate de tener acceso de administrador para usar todas las funciones de la ventana.
  • Revisa periódicamente el número de peregrinos y paradas para mantener la base de datos actualizada.
  • Utiliza los atajos de teclado para mejorar la eficiencia en las acciones comunes.
