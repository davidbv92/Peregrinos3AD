Ayuda - Ventana de Parada

En esta ventana, puedes gestionar la información sobre la parada seleccionada. Aquí puedes ver los datos de la parada, exportarlos o sellar el carnet de un peregrino.

Campos de Información

  • Nombre: Muestra el nombre de la parada.
  • Región: Muestra la región en la que se encuentra la parada.
  • Responsable: Muestra el responsable de la parada.

Botones de Acción

  • Mostrar Datos: Muestra información adicional sobre la parada en una ventana modal.
  • Sellar Carnet: Permite sellar el carnet de un peregrino, abriendo la ventana correspondiente para esta acción.
  • Exportar Datos: Permite exportar la información de la parada, aunque aún no se ha implementado un sistema de exportación.
  • Cerrar Sesión: Cierra la sesión del usuario y vuelve a la ventana de inicio de sesión.

Menú de Acciones Rápidas

  • Ctrl+M: Atajo para mostrar los datos de la parada.
  • Ctrl+X: Atajo para exportar los datos de la parada.
  • Ctrl+L: Atajo para sellar el carnet del peregrino.
  • Ctrl+A: Atajo para abrir esta ventana de ayuda.
  • Ctrl+S: Atajo para cerrar sesión y volver al login.

Errores comunes

  • Si los datos de la parada no se cargan correctamente, asegúrate de que el servicio de paradas está activo.
  • Si el botón "Sellar Carnet" no responde, verifica que el sistema está conectado correctamente y que los permisos para esta acción están configurados.
  • Si al intentar exportar los datos no sucede nada, puede ser que no haya un sistema de exportación configurado aún.

Consejos

  • Revisa que los datos de la parada estén completos y correctos antes de intentar exportarlos.
  • Utiliza los atajos de teclado para mejorar la eficiencia al gestionar las paradas.
  • Recuerda cerrar sesión cuando termines de trabajar para proteger tu información.
