Ayuda - Ventana del Peregrino

En esta ventana, puedes consultar y modificar los datos de tu cuenta de peregrino, así como exportar tu carnet en formato XML.

Campos de Información

  • Nombre: Muestra el nombre completo del peregrino.
  • Usuario: Muestra el nombre de usuario del peregrino.
  • Correo: Muestra la dirección de correo electrónico del peregrino.

Botones de Acción

  • Mostrar Datos: Muestra la información del peregrino en una ventana de alerta.
  • Modificar Datos: Permite editar los datos del peregrino. Abre la ventana para editar la información.
  • Exportar Carnet: Exporta los datos del peregrino y sus visitas a las paradas en un archivo XML. El archivo se guarda en el directorio src/main/resources/files con el nombre del peregrino seguido de "_peregrino.xml".
  • Cerrar Sesión: Cierra la sesión actual y vuelve a la ventana de inicio de sesión.

Menú de Acciones Rápidas

  • Ctrl+M: Atajo para mostrar los datos del peregrino.
  • Ctrl+D: Atajo para modificar los datos del peregrino.
  • Ctrl+X: Atajo para exportar el carnet del peregrino.
  • Ctrl+A: Atajo para abrir esta ventana de ayuda.
  • Ctrl+S: Atajo para cerrar sesión y volver al login.

Errores comunes

  • Si no se pueden exportar los datos del carnet, asegúrate de que el peregrino tiene al menos una parada y una estancia registrada.
  • Si los datos del peregrino no se muestran correctamente, verifica que el servicio de peregrinos esté funcionando correctamente.

Consejos

  • Asegúrate de que los datos del peregrino estén completos antes de exportar el carnet.
  • Utiliza los atajos de teclado para una navegación más rápida.
  • Cierra sesión correctamente para proteger tu información personal.
